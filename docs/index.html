What is Shadebot?

Shadebot is a general use Moderation and Welcoming bot. The bots main purpose is to welcome users from a specific room that type a password in, they will then be given a role and welcomed within a chosen room! Shadebot also does a few moderation things such as being able to mute users for a specific amount of time or mute/kick/ban a user for spamming mentions!
